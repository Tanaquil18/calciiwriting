    Section 1.2 The Method of Partial Fractions

    Objectives

    • How can we integrate rational functions when substitution does not work?
    • How do we break a rational function up into the sum of simpler rational functions?
    • What condition must be met in order to rewrite a rational function in this way?
    The need to integrate rational functions arises in differential equations and in other settings, and so we wonder about how to integrate them. If we are lucky, we can use substitution or integration by parts, but sometimes neither of those methods can be used.

    Example 1.2.1. Integrating A Rational Function with Substitution.

    Evaluate
    \begin{equation*} \int \frac{5x}{x^2+1} \, dx\text{.} \end{equation*}
    Solution.
    We see a function-derivative pair (up to a constant), since if \(u=x^2+1\) then \(du=2x\, dx\text{.}\) So we rewrite
    \begin{align*} \int \frac{5x}{x^2+1} \, dx \amp = \frac{5}{2}\int \frac{1}{u}\, du \\ \amp = \frac{5}{2}\ln|u|+C \\ \amp = \frac{5}{2}\ln|x^2+1|+C \text{.} \end{align*}

    Subsection 1.2.1 Starting the Method of Partial Fractions

    In Example¬†1.2.1, the derivative of the function in the denominator was sitting in the numerator (up to a constant multiple). What if we needed to integrate this instead?
    \begin{equation*} \int \frac{5x}{x^2-x-2} \, dx\text{?} \end{equation*}
    We could try what we did before and pick \(u=x^2-x-2\text{,}\) but then we calculate that \(du=(2x-1)\, dx\text{,}\) which does not help us in the numerator.
    Instead, we try a completely different strategy. If we somehow knew that
    \begin{equation*} \frac{5x}{x^2-x-2} = \frac{10/3}{x-2} + \frac{5/3}{x+1}\text{,} \end{equation*}
    then we could integrate both sides of the equation to get
    \begin{equation*} \int \frac{5x}{x^2-x-2} \, dx = \int \frac{10/3}{x-2} + \frac{5/3}{x+1} \, dx\text{,} \end{equation*}
    and the integral on the right is straightforward, giving us
    \begin{equation*} \int \frac{5x}{x^2-x-2} \, dx = \frac{10}{3} \ln|x-2| + \frac{5}{3}\ln|x+1| + C\text{.} \end{equation*}
    We can verify that
    \begin{equation*} \frac{5x}{x^2-x-2} = \frac{10/3}{x-2} + \frac{5/3}{x+1} \end{equation*}
    is indeed true by recombining the right-hand-side of the equation using a common denominator.
    \begin{align*} \frac{10/3}{x-2} + \frac{5/3}{x+1} \amp = \frac{10/3(x+1)}{(x-2)(x+1)} + \frac{5/3(x-2)}{(x+1)(x-2)}\\ \amp =\frac{\frac{10}{3}x+\frac{10}{3}+\frac{5}{3}x-\frac{10}{3} }{(x-2)(x+1)}\\ \amp =\frac{ \frac{15}{3}x}{(x-2)(x+1)}\\ \amp =\frac{5x}{x^2-x-2} \end{align*}
    The question is: how did we come up with
    \begin{equation*} \frac{5x}{x^2-x-2} = \frac{10/3}{x-2} + \frac{5/3}{x+1} \end{equation*}
    in the first place?
    One clue is that the denominator of the rational function in the integral factors into two linear terms,
    \begin{equation*} x^2-x-2 = (x-2)(x+1)\text{,} \end{equation*}
    and those are exactly the denominators of the simpler rational functions we used.
    We suppose that
    \begin{equation*} \frac{5x}{(x-2)(x+1)} = \frac{A}{x-2} + \frac{B}{x+1} \end{equation*}
    and then we can look for the constants \(A\) and \(B\text{.}\)
    Multiplying both sides of this equation by \((x-2)(x+1)\text{,}\) we find that
    \begin{equation*} \frac{5x\cancel{(x-2)(x+1)}}{\cancel{(x-2)(x+1)}}= \frac{A(x+1)\cancel{(x-2)}}{\cancel{x-2}} + \frac{B\cancel{(x+1)}(x-2)}{\cancel{x+1}} \end{equation*}
    which means
    \begin{equation*} 5x = A(x+1) + B(x-2)\text{.} \end{equation*}
    Because this equation should hold for every value of \(x\text{,}\) we can use inspired choices of specific \(x\)-values to help us find \(A\) and \(B\text{.}\) Taking \(x = -1\text{,}\) we have
    \begin{equation*} 5(-1) = A(0) + B(-3)\text{,} \end{equation*}
    so \(B = \frac{5}{3}\text{.}\) Choosing \(x = 2\text{,}\) it follows
    \begin{equation*} 5(2) = A(3) + B(0)\text{,} \end{equation*}
    so \(A = \frac{10}{3}\text{.}\) Thus,
    \begin{equation*} \int \frac{5x}{x^2-x-2} \, dx = \int \frac{10/3}{x-2} + \frac{5/3}{x+1} \, dx\text{.} \end{equation*}

    Subsection 1.2.2 When Can The Methods In This Example Be Used?

    Notice that any terms of the form \(\frac{A}{ax+b}\text{,}\) a number in the numerator divided by a linear term in the denominator, would be similarly easy to integrate, using \(u=ax+b\) so that \(du=a\, dx\text{.}\) Then
    \begin{equation*} \int \frac{A}{ax+b} \, dx = \frac{A}{a}\ln|ax+b|+C \end{equation*}
    The main question to explore in the next activity is: When can we integrate a rational function by rewriting into terms like this?

    Activity 1.2.2.

    1. Calculate the integral \(\displaystyle \int \frac{2x^2+3x+4}{x^3+x^2-2x}\, dx\) using the steps below.
      1. Factor the denominator, and use that factorization to write the general form of the partial fraction decomposition of the integrand, using letters like \(A, B, D\) etc., or \(A_1, A_2, A_3\text{,}\) etc.
      2. Calculate the antiderivatives of the partial fraction decomposition while still using letters like \(A, B, D\) etc., or \(A_1, A_2, A_3\text{,}\) etc.
      3. Finally, the least important step for calculus but often the most algebraically intense to do by hand, solve for the values of the letters you used, and write the final answer to the given integral.
    2. Consider the integral \(\displaystyle \int \frac{x^2+x+4}{(x^2+1)(x-1)}\, dx\text{.}\) The denominator is already factored, so we see what happens if we proceed like we did in part a.
      1. Calculate \(\int \frac{A}{x-1}dx+\int\frac{B}{x^2+1}dx\text{.}\)
      2. Suppose that \(\frac{x^2+x+4}{(x^2+1)(x-1)} = \frac{A}{x-1}+\frac{B}{x^2+1}\) and multiply both sides of the equation by \((x^2+1)(x-1)\) to obtain an equality with no fractions.
      3. Substitute \(x=1\) to solve for the value of \(A\text{,}\) and then substitute \(x=0\) to solve for the value of \(B\text{.}\) Does this result in an actual equality of the two sides? What equation involving \(A\) and \(B\) do you get if you substituted \(x=-1\text{?}\)
      4. Why does this show that \(\frac{x^2+x+4}{(x^2+1)(x-1)} \neq \frac{A}{x-1}+\frac{B}{x^2+1}\) for any values of \(A\) and \(B\text{?}\) How might we change the partial fraction decomposition form to fix this?
    Answer.
      1. \(x^3+x^2-2x=x(x^2+x-2)=x(x+2)(x-1)\text{,}\) which means that
        \begin{equation*} \frac{2x^2+3x+4}{x^3+x^2-2x} = \frac{A_1}{x}+\frac{A_2}{x+2}+\frac{A_3}{x-1} \end{equation*}
      2. \(\displaystyle \int \frac{2x^2+3x+4}{x^3+x^2-2x}dx = A_1\ln|x|+A_2\ln|x+2|+A_3\ln|x-1|+C\)
      3. \(\displaystyle \int \frac{2x^2+3x+4}{x^3+x^2-2x}dx = -2\ln|x|+\ln|x+2|+3\ln|x-1|+C\)
      1. \(\displaystyle A\ln|x-1|+B\arctan(x)+C\)
      2. \(x^2+x+4 = A(x^2+1)+B(x-1)\text{,}\) or \(x^2+x+4 = Ax^2+A+Bx-B\)
      3. Substituting \(x=1\) gives \(1^2+1+4=A(1^2+1)+B(1-1)\text{,}\) so \(A=3\text{.}\)
        Substituting \(x=0\) gives \(0^2+0+4=A(0^2+1)+B(0-1)\text{,}\) so \(4=A-B\text{,}\) and thus \(B=-1\text{.}\)
        Substituting \(x=-1\) gives \((-1)^2+(-1)+4=A((-1)^2+1)+B(-1-1)\text{,}\) or \(4=2A-2B\text{.}\)
      4. The equation \(4=2A-2B\) is not satisfied when \(A=3\) and \(B=-1\text{.}\) We could also see that we supposed
        \begin{equation*} x^2+x+4 = A(x^2+1)+B(x-1) \end{equation*}
        but the values we found would give us
        \begin{equation*} A(x^2+1)+B(x-1) = 3(x^2+1)-1(x-1) =3x^2-x+4 \end{equation*}
        and \(x^2+x+4 \neq 3x^2-x+4\) as functions. Try graphing them to see.

    Subsection 1.2.3 A Quadratic Term and a Linear Term

    As we saw in the activity, we have to be careful with the form of our partial fraction decomposition if factoring the denominator results in anything other than distinct linear terms.
    For example, when we had a numerator of \(x^2+x+4\text{,}\) there are \(3\) numbers: the coefficient of the \(x^2\) term, the coefficient of the \(x\) term, and the constant term. Using \(\frac{A}{x-1}+\frac{B}{x^2+1}\) gave us \(2\) numbers to solve for, \(A\) and \(B\text{,}\) and in this case, it was impossible to find values of \(A\) and \(B\) to make the equality hold. But we can modify the partial fraction form to introduce a third number.

    Example 1.2.2. Integrating A Rational Function with Substitution.

    Evaluate
    \begin{equation*} \int \frac{x^2+x+4}{(x^2+1)(x-1)} \, dx\text{.} \end{equation*}
    We suppose that
    \begin{equation*} \frac{x^2+x+4}{(x^2+1)(x-1)} = \frac{A}{x-1} + \frac{Bx+D}{x^2+1} \end{equation*}
    and note that this gives us \(3\) constants to work with.‚Äâ1‚Äâ Before we look for the values of the constants \(A\text{,}\) \(B\) and \(D\text{,}\) we will integrate to make sure this is a better form than the one we started with.
    An antiderivative of \(\frac{A}{x-1}\) is \(A\ln|x-1|\text{.}\) We can calculate
    \begin{equation*} \int \frac{Bx+D}{x^2+1} \, dx = \int \frac{Bx}{x^2+1} \, dx + \int \frac{D}{x^2+1} \, dx \end{equation*}
    and note that the second integral is one of our basic forms, \(D\arctan(x)\text{,}\) while the first can be evaluated with substitution taking \(u=x^2+1\) so that \(du=2x\, dx\text{.}\)
    Thus we see that if we can find such numbers \(A\text{,}\) \(B\) and \(D\text{,}\) then we can integrate
    \begin{align*} \int \frac{x^2+x+4}{(x^2+1)(x-1)} \, dx \amp = \int \left(\frac{A}{x-1} + \frac{Bx+D}{x^2+1}\right) dx\\ \amp = \int \frac{A}{x-1} dx + \int \frac{Bx}{x^2+1}dx + \int \frac{D}{x^2+1}dx\\ \amp = A\ln|x-1|+\frac{B}{2}\ln|x^2+1|+D\arctan(x) +C \end{align*}
    Now that we‚Äôve done the calculus part, we turn our attention to the algebra. Multiplying both sides of the integrand by \((x-1)(x^2+1)\text{,}\) we find that
    \begin{equation*} \frac{(x^2+x+4)\cancel{(x-1)(x^2+1)}}{\cancel{(x-1)(x^2+1)}}= \frac{A(x^2+1)\cancel{(x-1)}}{\cancel{x-1}} + \frac{(Bx+D)\cancel{(x^2+1)}(x-1)}{\cancel{x^2+1}} \end{equation*}
    which means
    \begin{equation*} x^2+x+4 = A(x^2+1) + (Bx+D)(x-1)\text{.} \end{equation*}
    We have several tools in our toolbox for solving for the unknown numbers. We could...
    • ... use the black box method. Use a solver like Wolfram Alpha or another computational tool.
    • ... set \(x\) equal to different values like we‚Äôve done before. This is slightly more complicated than when the denominator factored into linear terms, but we can still do it, taking \(x=1\text{,}\) then \(x=0\text{,}\) then \(x=-1\text{,}\) or any three values of \(x\) you choose and find easiest to work with.
    • ... compare coefficients of both sides of the equation and get a system of equations to solve.
    Since the first two methods don‚Äôt give us anything new to discuss, we choose to demonstrate the last method here. Recall that we are trying to solve
    \begin{equation*} x^2+x+4 = A(x^2+1) + (Bx+D)(x-1) \end{equation*}
    for the unknown values of \(A\text{,}\) \(B\) and \(D\) so that the two sides are equal for all possible values of \(x\text{.}\) This means the two sides must really be the same function of \(x\text{,}\) or the same polynomial of \(x\text{.}\) Multiplying out the right-hand side and grouping like terms together gives
    \begin{align*} x^2+x+4 \amp = A(x^2+1) + (Bx+D)(x-1) \\ \amp = Ax^2+A + Bx^2-Bx+Dx-D\\ \amp = (A+B)x^2+(-B+D)x+(A-D) \end{align*}
    Setting like coefficients equal means that \(A+B=1\) (because those are the corresponding coefficients of the \(x^2\) term) and \(-B+D=1\) (because those are the corresponding coefficients of the \(x\) term) and \(A-D=4\) (because those are the constant terms).
    This gives us a system of linear equations, three equations and three unknowns. Combining this with the second method, of setting \(x=1\) to find that \(1^2+1+4=A(1^2+1)+(Bx+D)(1-1)\) and so \(A=3\) would reduce this to two unknowns and often be the fastest to solve by hand. (Since \(A=3\) and \(A+B=1\text{,}\) this means that \(B=-2\text{;}\) similarly, since \(A-D=4\text{,}\) this means that \(D=-1\text{.}\)) Since we already integrated to find that
    \begin{equation*} \int \frac{x^2+x+4}{(x^2+1)(x-1)} \, dx = A\ln|x-1|+\frac{B}{2}\ln|x^2+1|+D\arctan(x) +C \end{equation*}
    we can substitute the values we‚Äôve found and calculate that
    \begin{equation*} \int \frac{x^2+x+4}{(x^2+1)(x-1)} \, dx = 3\ln|x-1|+\frac{-2}{2}\ln|x^2+1|-1\arctan(x) +C \end{equation*}
    which answers the question we were attempting to solve.
    However, it‚Äôs worth noting that if we‚Äôve had a little bit of matrix theory or linear algebra, we can use matrices to solve a system of linear equations, and that would give us
    \begin{align*} \amp A \amp \amp+ B \amp \amp \ \amp \amp=1 \\ \amp \amp \amp- B \amp \amp+ D \amp \amp=1 \\ \amp A \amp \amp \amp \amp-D \amp \amp=4 \end{align*}
    which becomes \(\begin{bmatrix} 1 \amp 1 \amp 0 \amp 1 \\ 0 \amp -1 \amp 1 \amp 1 \\1 \amp 0 \amp -1 \amp 4 \end{bmatrix}\text{.}\)
    Solving this using Gaussian elimination results in \(\begin{bmatrix} 1 \amp 0 \amp 0 \amp 3 \\ 0 \amp 1 \amp 0 \amp -2 \\0 \amp 0 \amp 1 \amp -1 \end{bmatrix}\text{,}\) which does give us the same values of \(A=3\text{,}\) \(B=-2\text{,}\) and \(D=-1\text{.}\)
    However, this matrix form tells us something more: that this system of equations will have a unique solution regardless of the starting coefficients of the quadratic in the numerator of the integrand.
    Note that we don‚Äôt have to use matrices at all - we have other tools for solving a system of equations. However, this example generalizes and bears repeating: as long as the numerator of the integrand is less than or equal to degree \(2\text{,}\) and the denominator factors into a linear and an irreducible quadratic, we can find a partial fraction decomposition and calculate the integral by setting it up in this way, with a constant over the linear factor and a linear term over the quadratic factor.‚Äâ2‚Äâ
    Furthermore, we can see the problem that occurred when we had the wrong form of the partial fraction decomposition. When we attempted to set \(\frac{x^2+x+4}{(x^2+1)(x-1)} = \frac{A}{x-1}+\frac{B}{x^2+1}\) and cleared the denominators, we got
    \begin{equation*} x^2+x+4 = A(x^2+1)+B(x-1) = Ax^2+Bx+A-B. \end{equation*}
    Setting corresponding coefficients equal gives us
    \begin{align*} \amp A \amp \amp \amp \amp=1 \\ \amp \amp \amp B \amp \amp=1 \\ \amp A \amp \amp-B \amp \amp=4 \end{align*}
    which becomes \(\begin{bmatrix} 1 \amp 0 \amp 1 \\ 0 \amp 1 \amp 1 \\1 \amp -1 \amp 4 \end{bmatrix}\text{,}\) which row reduces to \(\begin{bmatrix} 1 \amp 0 \amp 1 \\ 0 \amp 1 \amp 1 \\0 \amp 0 \amp 1 \end{bmatrix}\text{,}\) an inconsistent system (no possible solutions for \(A\) and \(B\text{.}\))
    One of the important takeaways from this is that the form of the partial fraction decomposition matters.

    Subsection 1.2.4 General Method of Partial Fractions

    Let‚Äôs collect and summarize what we can say about the possible forms of partial fraction decompositions.

    Partial Fraction Decomposition.

    Given any rational function with the degree of the numerator strictly less than the degree of the denominator, we can rewrite the function as a sum of simpler terms.
    Denominator Contains Partial Fraction Decomposition Contains
    \((ax+b)\) \(\frac{A}{ax+b}\)
    \((ax+b)^n\) \(\frac{A_1}{ax+b}+\frac{A_2}{(ax+b)^2}+\cdots + \frac{A_n}{(ax+b)^n}\)
    \((ax^2+bx+c)\) \(\frac{Ax+B}{ax^2+bx+c}\)
    \((ax^2+bx+c)^n\) \(\frac{A_1x+B_1}{ax^2+bx+c}+\frac{A_2x+B_2}{(ax^2+bx+c)^2}+\cdots + \frac{A_nx+B_n}{(ax^2+bx+c)^n}\)
    This can be stated in words in the following way.
    • When the factored form of the denominator contains a linear term, the partial fraction decomposition contains a constant over that linear term.
    • When the factored form of the denominator contains a power of a linear term, the partial fraction decomposition contains a constant over every power of that linear term up to the max power.
    • When the factored form of the denominator contains a quadratic term, the partial fraction decomposition contains a linear term divided by the quadratic term.
    • When the factored form of the denominator contains a power of a quadratic term, the partial fraction decomposition contains a linear term over every power of that quadratic term up to the max power.

    Checkpoint 1.2.3. Don‚Äôt Forget to Factor.

      What is the partial fraction decomposition of \(\frac{5x-2}{x^2-1}\text{?}\)
    • \(\frac{A}{x^2-1}\)
    • This would be true if the denominator was linear instead of quadratic.
    • \(\frac{Ax+B}{x^2-1}\)
    • This would be true if the denominator was an unfactorable quadratic, but the denominator can be factored.
    • \(\frac{A}{x+1}+\frac{B}{x-1}\)
    • Correct!

    Checkpoint 1.2.4. Powers.

      What is the partial fraction decomposition of \(\frac{5x-2}{(x+2)^2}\text{?}\)
    • \(\frac{B}{x+2}\)
    • This would be true if the exponent of the original denominator was 1 instead of 2.
    • \(\frac{Ax+B}{(x+2)^2}\)
    • This would be true if the denominator was an unfactorable quadratic instead of a power of a linear term.
    • \(\frac{A}{(x+2)^2}\)
    • This is not enough terms to account for all the possible original numerators when the denominator has a power of a linear term.
    • \(\frac{A}{x+2}+\frac{B}{(x+2)^2}\)
    • Correct!

    Checkpoint 1.2.5. Powers again.

      What is the partial fraction decomposition of \(\frac{5}{(x+2)^3}\text{?}\)
    • \(\frac{A}{x+2}+\frac{B}{(x+2)^2}+\frac{D}{(x+2)^3}\)
    • Technically correct, but not the best choice here. Look at the original numerator again.
    • \(\frac{A}{(x+2)^3}\)
    • Correct! And in fact, A=5. (If this needed to be integrated, we can use the power rule straightaway, no partial fractions work is required.)

    Activity 1.2.4.

    In this activity, we will calculate

    Subsection 1.2.6 Final Considerations

    An especially astute reader might notice that we said that the denominator can contain terms of the form \(ax^2+bx+c\) but we only considered terms of the form \(x^2+k\) in our examples and activities. The reason is that every \(ax^2+bx+c\) can be written as \(a\left( \left(x-\frac{b}{2a}\right)^2+k\right)\) if we recall how to complete the square from algebra class, and then using a substitution ‚Äâ3‚Äâ turns this into some constant number multiplied by \(y^2+k\text{.}\)
    But this turns a calculus problem into even more of an algebra problem, and the focus of this material is integration, not algebra.
    You might also be wondering what happens if the denominator doesn‚Äôt factor into powers of linear and quadratic terms. It is a theorem beyond the scope of this class, called the Fundamental Theorem of Algebra, that every polynomial over the real numbers factors into powers of linear and quadratic terms.
    The result is that every rational function, regardless of the degree of the numerator or denominator, or how the denominator factors, can be integrated using some combination of the techniques described in this section. Putting that theory into practice, however, is very often no trivial matter!

    Subsection 1.2.7 Summary

    • Given any rational function with the degree of the numerator strictly less than the degree of the denominator, we can rewrite the function as a sum of terms of the form
      \begin{equation*} \frac{A}{(ax+b)^n}\ \text{or }\frac{Ax+B}{\left(ax^2 + bx +c \right)^n} \end{equation*}
      where \(A\text{,}\) \(B\text{,}\) \(a\text{,}\) \(b\text{,}\) and \(c\) are real numbers, and \(n\) is a natural number.
    • Which of the forms are included in the decomposition of a rational function depends entirely on the factored form of the denominator, and each of those forms can be integrated using basic integrals, substitution, and/or trigonometric substitution, possibly requiring some algebra.
    • Similar to how every improper fraction can be expressed as an integer plus a proper fraction, every rational function with the degree of the numerator greater than or equal to the degree of the denominator can be expressed as \(p(x)+\frac{q(x)}{r(x)}\) where \(p(x)\) is a polynomial and the degree of \(q(x)\) is strictly less than the degree of \(r(x)\text{.}\)
    • This means that every rational function has an antiderivative function we can express without an integral.
    We avoid using \(C\) as a coefficient because we don‚Äôt want to get it confused with the \(+C\) at the end of an indefinite integral.
    See later for what to do if the degree is greater than or equal to \(3\) with this same factored form of the denominator.
