Activity 1.2.2.

  1. Calculate the integral \(\int \frac{2x^2+3x+4}{x^3+x^2-2x}\, dx\) using the steps below.
    1. Factor the denominator, and use that factorization to write the general form of the partial fraction decomposition of the integrand, using letters like \(A, B, D\) etc., or \(A_1, A_2, A_3\text{,}\) etc.
    2. Calculate the antiderivatives of the partial fraction decomposition while still using letters like \(A, B, D\) etc., or \(A_1, A_2, A_3\text{,}\) etc.
    3. Finally, the least important step for calculus but often the most algebraically intense to do by hand, solve for the values of the letters you used, and write the final answer to the given integral.
  2. Consider the integral \(\int \frac{x^2+x+4}{(x^2+1)(x-1)}\, dx\text{.}\) The denominator is already factored, so we see what happens if we proceed like we did in part a.
    1. Calculate \(\int \frac{A}{x-1}dx+\int\frac{B}{x^2+1}dx\text{.}\)
    2. Suppose that \(\frac{x^2+x+4}{(x^2+1)(x-1)} = \frac{A}{x-1}+\frac{B}{x^2+1}\) and multiply both sides of the equation by \((x^2+1)(x-1)\) to obtain an equality with no fractions.
    3. Substitute \(x=1\) to solve for the value of \(A\text{,}\) and then substitute \(x=0\) to solve for the value of \(B\text{.}\) Does this result in an actual equality of the two sides? What equation involving \(A\) and \(B\) do you get if you substituted \(x=-1\text{?}\)
    4. Why does this show that \(\frac{x^2+x+4}{(x^2+1)(x-1)} \neq \frac{A}{x-1}+\frac{B}{x^2+1}\) for any values of \(A\) and \(B\text{?}\) How might we change the partial fraction decomposition form to fix this?
