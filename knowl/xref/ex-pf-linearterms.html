Example 1.2.2. Two linear terms.

Show that
\begin{equation*} \int \frac{5x}{x^2-x-2} \, dx = \frac{10}{3} \ln|x-2| + \frac{5}{3}\ln|x+1| + C\text{.} \end{equation*}
Answer: We first factor the denominator \(x^2-x-2 = (x-2)(x+1)\) and then suppose that
\begin{equation} \frac{5x}{(x-2)(x+1)} = \frac{A}{x-2} + \frac{B}{x+1}\tag{1.2.2} \end{equation}
for some constants \(A\) and \(B\text{.}\) If this is true, the integral becomes much easier to calculate, since we can integrate both sides of (1.2.2) to get
\begin{equation} \int \frac{5x}{x^2-x-2} \, dx = A \ln|x-2| + B\ln|x+1| + C\text{.}\tag{1.2.3} \end{equation}
We now start the process of solving for the values of \(A\) and \(B\text{.}\) Multiplying both sides of (1.2.2) by the integrandâ€™s denominator \((x-2)(x+1)\text{,}\) we find that
\begin{equation*} \frac{5x\cancel{(x-2)(x+1)}}{\cancel{(x-2)(x+1)}}= \frac{A(x+1)\cancel{(x-2)}}{\cancel{x-2}} + \frac{B\cancel{(x+1)}(x-2)}{\cancel{x+1}} \end{equation*}
which means
\begin{equation} 5x = A(x+1) + B(x-2)\text{.}\tag{1.2.4} \end{equation}
From here, we show one possible method of solving for the values of \(A\) and \(B\text{.}\)
Because (1.2.4) is true for every value of \(x\text{,}\) we can use inspired choices of specific \(x\)-values to help us find \(A\) and \(B\text{.}\)
Taking \(x = -1\text{,}\) we have
\begin{equation*} 5(-1) = A(0) + B(-3)\text{,} \end{equation*}
so \(B = \frac{5}{3}\text{.}\)
Now setting \(x = 2\text{,}\) it follows that
\begin{equation*} 5(2) = A(3) + B(0)\text{,} \end{equation*}
so \(A = \frac{10}{3}\text{.}\)
Thus, substituting the values of \(A\) and \(B\) into (1.2.3) we get
\begin{equation*} \int \frac{5x}{x^2-x-2} \, dx = \frac{10}{3}\ln|x-2| + \frac{5}{3}\ln|x+1|+C\text{.} \end{equation*}
